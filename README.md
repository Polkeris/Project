## Интернет-магазин на Django (SQLite)

### Установка зависимостей

Требуется установленный Python (у нас используется Python 3.12 через `uv`).

```powershell
uv sync
```

### Запуск миграций

```powershell
uv run python manage.py migrate
```

### Создание суперпользователя (админка)

```powershell
uv run python manage.py createsuperuser
```

### Запуск сервера

```powershell
uv run python manage.py runserver
```

Открыть:

- Главная страница: `http://127.0.0.1:8000/`
- Админка: `http://127.0.0.1:8000/admin/`

### Как наполнить магазин товарами

1) Зайди в админку и создай категории.
2) Создай товары и укажи им категорию (и при желании загрузить изображение).
3) На главной появится список товаров, корзина работает через сессии.
